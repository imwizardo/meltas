О нас

Мы — команда экспертов, объединённых идеей создать надёжную базу данных мошенников. Наш бот отслеживает подозрительную активность, проверяет репутацию пользователей и помогает оперативно реагировать на случаи скама. С нами вы всегда в курсе последних новостей и можете быть уверены в безопасности сделок.

Функции бота

Проверка репутации

Команда /чек позволяет мгновенно узнать репутацию пользователя и получить полную статистику (слито скамеров, предупреждения и т.д.).

Добавление скамеров

С помощью команды /скам можно оперативно добавить мошенника в базу с подробным описанием и ссылками на доказательства.

Модерация и уведомления

Бот автоматически рассылает напоминания, генерирует отчёты и предоставляет инструменты модерации (бан, мут, апелляция и т.д.).

Премиум-функции

Пользователи с премиум-статусом получают расширенные возможности: смена темы, кастомное фото, увеличение лимитов проверок и многое другое.

Бот MELTAS

Наш бот разработан на Python с использованием asyncio, aiohttp, aiosqlite и Pyrogram. Он интегрируется с Telegram и реализует множество функций: от проверки репутации до автоматической модерации и рассылок. Бот поддерживает команды для пользователей, модераторов и владельцев, что позволяет оперативно реагировать на случаи скама.

Ключевые команды

/start

Запуск бота в личных сообщениях.

/чек

Быстрая проверка репутации по ID или @username.

/скам

Добавление мошенника в базу с подробным описанием и ссылками на пруфы.

/admin, /гарант, /президент

Команды для назначения ролей и модерации пользователей.

/tips, /simulate

Советы по безопасности и симулятор безопасных сделок.

Пример кода

import asyncio
from pyrogram import Client, filters
import aiosqlite

async def main():
    # Инициализация базы данных
    db = await aiosqlite.connect("bot_data.db")
    # Создание и настройка клиента бота
    app = Client("my_bot", api_id=API_ID, api_hash=API_HASH, bot_token=BOT_TOKEN)
    
    @app.on_message(filters.command("start"))
    async def start(client, message):
        await message.reply("Привет! Я MELTAS бот для борьбы со скамом.")
    
    # Запуск бота
    await app.start()
    await idle()
    await app.stop()
    await db.close()

if __name__ == "__main__":
    asyncio.run(main())

Дополнительная информация о боте

Наш бот – это многофункциональный инструмент, который позволяет:

  • Отслеживать и регистрировать мошенническую активность в режиме реального времени.
  • Обеспечивать оперативную модерацию групп: выдача мутов, банов, автоматическая рассылка предупреждений и уведомлений.
  • Проводить ежедневные и всеобщие отчёты по сливу скамеров, что помогает выявлять самых активных участников.
  • Интегрировать внешние сервисы, например, TikTokApi для получения трендовых видео.
  • Обрабатывать inline-запросы для мгновенной проверки репутации пользователей.
  • Предоставлять расширенные возможности для премиум-пользователей: кастомизация профиля, смена темы, увеличение лимитов проверок.
  • Поддерживать функционал для управления ролями: администраторы, гаранты, президенты, директора и стажёры.
